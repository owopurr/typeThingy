## setup
download python<br>
`pip install -r requirements.txt`<br>
make a new folder called `fonts`<br>
put your .ttf files inside<br>
in `config.json`, add your fonts like so:<br>
```json
"fontname":"./fonts/file.ttf"
```
<br>
(you'd use `fontname` like `fnt:fontname` in the script)<br>
change script<br>
`py main.py`<br>
`output.avi` is your video!
